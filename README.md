# SET-WEB-RAILS

## Description(詳細)

シス研メンバーでRailsでWebサービスを作成する

## Requirement(必要条件)

|言語/FW|Version|
|:--|--:|
|ruby|2.5.3|
|Rails|5.2.1|
|docker|18.06.1-ce|
|docker-compose|1.22.0|

## Usage(使い方)

**`make help` で何が出来るかを表示出来ます**

1. `make`
2. access [http://localhost:3000/](http://localhost:3000)

---

2回目以降

1. `make run`
2. access [http://localhost:3000/](http://localhost:3000)
